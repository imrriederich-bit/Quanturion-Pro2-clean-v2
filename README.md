# Quanturion Pro – Refactor

Dieses Paket ist eine bereinigte, strukturierte Fassung deiner hochgeladenen App.

## Neu angelegt
- `src/utils/` – Hilfsfunktionen für Zahlen, Labels und Formatierung
- `src/services/` – Rechenlogik und WhatsApp-Weiterleitung
- `src/data/` – Tarife, Provider und Preview-Daten
- `src/i18n/` – Texte für DE / EN / HE
- `src/screens/` – Dashboard, Modul, Premium, Legal
- `src/components/` – wiederverwendbare UI-Bausteine
- `src/styles/` – Theme und globale Styles

## Wichtige Hinweise
- Die alten Dateien liegen zur Sicherheit unter `legacy/`.
- Ein Logo-Platzhalter ist jetzt unter `assets/logo-pro.png` eingebunden.
- Wenn du dein finales Original-Logo nutzen willst, ersetze einfach diese Datei durch deine endgültige PNG-Datei mit demselben Namen.

## Start
```bash
npm install
npm start
```
